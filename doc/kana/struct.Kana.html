fn half2full(&self, s: &str) -> String

Convert Half-width-kana into normal Katakana with diacritical marks separated [ｱﾞﾊﾟ -> ア゙パ]

This method is simple, but tends to cause troubles when rendering. In such a case, use half2kana() or execute vsmark2{half|full|combi}() as a post process.

Examples

use kana::Kana;
let k = Kana::init();
let s1 = "ﾏﾂｵ ﾊﾞｼｮｳ ｱﾟ";
assert_eq!("マツオ バショウ ア゚", k.half2full(s1));
